# Kernel Embeddings and the Separation of Measure Phenomenon

This repository contains the implementation and analysis for the paper **"Kernel Embeddings and the Separation of Measure Phenomenon"** by Leonardo V. Santoro, Kartik G. Waghmare, and Victor M. Panaretos.
It provides the code and resources used to explore the separation of measure phenomenon using kernel embeddings, as described in the paper, and replicate the experiments and plots therein.

## Usage
1. Clone the repository:
    ```bash
    git clone https://github.com/your-repo/KernelSeparation--PNAS.git
    cd KernelSeparation--PNAS
    ```

2. Run the main script:
    ```bash
    python main.py
    ```

3. The results will be saved as `AR1.png` in the working directory.
